# Suraiyya Landing Page

Standalone landing page for Suraiyya - ADHD Productivity App for Neurodivergent Adults.

## Overview

This is a separate, standalone landing page that redirects users to the main app at https://suraiyya.vercel.app/ for login/signup.

## Files

- `index.html` - Complete landing page with all CSS and JS inline
- `s_logo.png` - Purple Suraiyya logo
- `README.md` - This file

## Deployment

This is a static HTML page that can be deployed anywhere:

### Option 1: Vercel
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel --prod
```

### Option 2: Netlify
```bash
# Install Netlify CLI
npm i -g netlify-cli

# Deploy
netlify deploy --prod --dir .
```

### Option 3: GitHub Pages
1. Push this folder to a GitHub repository
2. Go to Settings > Pages
3. Select source as main branch
4. Your site will be live at `https://yourusername.github.io/repo-name`

### Option 4: Any Static Hosting
Simply upload these files to any static hosting service (AWS S3, Firebase Hosting, Cloudflare Pages, etc.)

## Configuration

To change the app URL that login/signup buttons redirect to, edit the href attributes in `index.html`:

Search for `https://suraiyya.vercel.app` and replace with your app's URL.
